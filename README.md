# Medical RAG Chatbot

A Retrieval-Augmented Generation (RAG) Medical Chatbot built using:

- LangChain
- ChromaDB
- HuggingFace Embeddings
- Groq LLM
- Streamlit

## Features

- Medical question answering
- PDF knowledge base
- Vector search using ChromaDB
- Streaming responses
- ChatGPT-like UI

## Run

pip install -r requirements.txt

streamlit run medibot.py
